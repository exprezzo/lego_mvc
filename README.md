lego_mvc
========

arquitectura MVC modular, como piezas Lego

========
Historias de usuario:

Quiero una arquitectura MVC ligera y sencilla.
Quiero que sea construida usando TDD.
	* Realizar pruebas unitarias y del funcionamiento de la arquitectura
Quiero que se consideren los nice-urls
Quiero un cargador de módulos
Quiero un modulo para hacer pruebas unitarias desde una interfaz web, 
	*que muestre la lista de pruebas, y la prueba seleccionada con un boton ejecutar prueba y el resultado de la prueba con colores
